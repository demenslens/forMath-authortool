ForMath - Wiskundige Expressie Visualizer
==========================================

ForMath parseert wiskundige expressies, genereert een AST (Abstract Syntax Tree)
visualisatie als SVG, en exporteert een gedetailleerde JSON beschrijving met
mathblocks, steps en DUO verzameling.


VEREISTEN
---------
- Python 3.8 of hoger (geen extra packages nodig)
- Een moderne webbrowser (Chrome, Firefox, Safari, Edge)


MAPSTRUCTUUR
------------
Na het uitpakken van het zip-bestand:

  formath/
  ├── formath_web/
  │   ├── server.py            ← Webserver (start hier)
  │   ├── index.html           ← HTML-structuur van de invoerpagina
  │   ├── app.css              ← Styling
  │   └── app.js               ← Frontend-logica (MathLive, zoom, API)
  ├── python_bestanden/
  │   ├── expression_parser.py ← Expressie parser (LaTeX → AST)
  │   ├── ast_normalizer.py    ← AST normalisatie
  │   ├── manifold_detector.py ← Manifold keten detectie
  │   ├── manifold_converter.py← Manifold conversie
  │   ├── ast_visualizer.py    ← SVG visualisatie
  │   └── json_exporter.py     ← JSON export
  ├── _archief/                ← Oude/ongebruikte modules (niet nodig om te draaien)
  └── INSTALLATIE.txt          ← Dit bestand


STARTEN
-------

1. Open een terminal (Mac: Terminal, Windows: Command Prompt of PowerShell)

2. Navigeer naar de formath_web map:

   Mac/Linux:
     cd pad/naar/formath/formath_web

   Windows:
     cd pad\naar\formath\formath_web

3. Start de server:

   Mac/Linux:
     python3 server.py

   Windows:
     python server.py

4. Je ziet in de terminal:
     [OK] ForMath AST pipeline geladen
     Server draait op http://localhost:8765

5. Open in je browser:
     http://localhost:8765


GEBRUIK
-------
- Typ een wiskundige expressie in het invoerveld (MathLive editor).
  Voorbeelden:
    1/2 + 1/3
    (1+2) x (3+4) + 5
    sqrt(16)              (vierkantswortel)
    \sqrt[3]{27}          (derdemachtswortel)
    2^3 + 5               (machtsverheffen)

- Klik op "OK" om de AST boom als SVG te genereren.
- Klik op "JSON" om de uitgebreide JSON en SVG te exporteren.

De JSON en SVG bestanden worden opgeslagen in:
  ~/Desktop/formath_JSON/    (Mac/Linux)
  %USERPROFILE%\Desktop\formath_JSON\    (Windows)

Met bestandsnamen als:
  opgave_20260311_001.json
  opgave_20260311_001.svg


STOPPEN
-------
Druk op Ctrl+C in de terminal om de server te stoppen.


OPMERKINGEN
-----------
- Vermenigvuldiging: *, × (MathLive \times) of \cdot — NIET de punt (.).
- Deling: / voor gewone breuken, : voor de delingsoperator.
- Breuken: typ 1/2 of gebruik \frac{1}{2} in LaTeX.
- Vierkante haken [ ] werken als haakjes.
- Het wortelteken: \sqrt{x} voor vierkantswortel, \sqrt[n]{x} voor n-de wortel.
